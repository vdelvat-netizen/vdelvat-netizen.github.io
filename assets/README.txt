HOW TO ADD YOUR PHOTO AND REAL LOGOS
=====================================

The site works fine without any files here — it shows tasteful text
placeholders. Drop the files below in and they appear automatically,
with zero code changes.

1) YOUR HERO PHOTO  (shows next to your name)
   - Put a file named exactly:  assets/victor.jpg
   - Recommended: a portrait-orientation photo, ~800 x 1000px (4:5),
     face roughly centred. .jpg or .png both fine (keep the name .jpg).

2) WORK / SCHOOL LOGOS  (small badge next to each entry)
   Put official logo files (SVG preferred, or PNG with transparent
   background) in assets/logos/ using these exact names:

     assets/logos/edhec.svg        (EDHEC Business School)
     assets/logos/ucla.svg         (UCLA Extension)
     assets/logos/ntu.svg          (Nanyang Technological University)
     assets/logos/nicepotager.svg  (#NicePotager — your own project)
     assets/logos/scouts.svg       (Scouts de France)
     assets/logos/innout.svg       (In-N-Out)

   If a file is missing, the site simply shows the name as text — no error.

Tip: official logos are usually downloadable from each organisation's
"press" / "brand" / "media kit" page. Use the SVG or transparent PNG.
